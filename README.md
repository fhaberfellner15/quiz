# Career Quiz

Ein interaktives Quiz, das herausfindet, welcher IT-Beruf am besten zu dir passt.

## Features

- 8 zufällig ausgewählte Fragen aus einem Pool von 30+
- 8 Karrierepfade: Softwareentwicklung, UX/UI Design, Projektmanagement, Data Analytics, Marketing, DevOps, Unternehmensberatung, HR
- Antworten werden bei jeder Runde neu gemischt
- Ergebnis zeigt die zwei passendsten Karrieren mit Beschreibung

## Tech Stack

- [Vue 3](https://vuejs.org/) + Composition API
- [TypeScript](https://www.typescriptlang.org/)
- [Pinia](https://pinia.vuejs.org/) (State Management)
- [Bootstrap 5](https://getbootstrap.com/)
- [Vite](https://vite.dev/)

## Setup

```bash
npm install
npm run dev
```
